### /

Главная страница. Можно выбрать корпус, этаж и аудиторию и увидеть их на карте МФТИ.

### /login

Вход в аккаунт.

### /register

Регистрация.

### /profile

Профиль.

### /schedule

Расписание. Можно добавлять и удалять предметы, а также переходить на карту для отображения их аудиторий.
